## 작업 규칙

1. 요청한 작업은 한번에 모두 처리할 것
2. 수정이 필요한 파일만 최소한으로 읽을 것
3. 불필요한 파일 읽기 금지
4. 확인 질문 최소화하고 바로 실행할 것
5. 작업 완료 후 변경된 파일 목록만 간단히 보고
6. 항상 Shift+Tab auto-accept 모드로 진행
7. 코드 설명은 짧게, 실행 위주로

## 토큰 절약 규칙

1. 한번에 여러 수정사항 묶어서 처리
2. 중복 작업 하지 말 것
3. 이미 본 파일 다시 읽지 말 것
4. 불필요한 주석 달지 말 것

## 프로젝트 기본 정보

- 프레임워크: Next.js (App Router)
- 스타일: Tailwind CSS
- 주요 파일: app/page.tsx
- 배포: Vercel (자동 배포)
- 타겟: 한국에 관심있는 외국인
- 언어: 영어
